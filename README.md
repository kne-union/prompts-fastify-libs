# @kne/prompts-fastify-libs 文档索引

本项目包含多个文档，用于 Fastify 业务插件开发过程中各类 AI 辅助任务的提示词模板。

## 文档列表

### 1. Fastify-Account 使用指南

**功能**: 用户账号管理插件，提供完整的用户注册、登录、认证、密码管理等功能，支持邮箱和手机号两种登录方式。

**适用场景**: 
- 需要用户认证功能的 Fastify 应用开发
- 用户管理系统开发
- 集成 JWT 认证机制
- 实现验证码登录流程

**核心内容**:
- JWT Token 认证与中间件
- 验证码发送与验证系统
- 用户注册、登录、密码管理 API
- 管理员功能与权限控制
- 服务层方法详解

**使用方式**: 阅读 [原文档](prompts/fastify-account使用指南.md) 了解详情

### 2. Fastify-Message 使用指南

- 发送邮件通知和系统消息
- 短信验证码发送
- 批量消息推送
- 消息模板管理

**核心内容**:
- EJS 模板系统
- 邮件发送配置
- 自定义发送器扩展
- 消息记录追踪

**使用方式**: 阅读 [原文档](prompts/fastify-message使用指南.md) 了解详情
